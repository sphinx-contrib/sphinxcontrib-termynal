Usage
=====

**termynal** usage documentation.

Installation
------------

Install **termynal** directly from pypi using pip:

.. code-block::

    pip install sphinxcontrib-termynal

To activate the extension you must add it to your ``conf.py`` file:

.. code-block:: python

    # docs/conf.py

    extensions = [
        ...
        "sphinxcontrib.termynal",
        ...
    ]
